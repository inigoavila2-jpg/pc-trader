## Troubleshooting File Issues

### Issue: "Cannot find module '@/components/AIAgentChatbox'"

**Cause**: File not in correct location

**Check**:
```bash
dir src\components\AIAgentChatbox.jsx
```

If missing, copy the file to `src/components/`

### Issue: "import useChatHistory failed"

**Cause**: Hook file not found

**Check**:
```bash
dir src\hooks\useChatHistory.js
```

If missing, create `src/hooks/` folder and copy the file.

### Issue: "VITE_GEMINI_KEY undefined"

**Cause**: `.env` file missing or not loaded

**Check**:
1. Does `.env` exist in project root?
   ```bash
   dir .env
   ```
2. Does it have `VITE_GEMINI_KEY=...`?
   ```bash
   type .env
   ```
3. Restart dev server after creating `.env`:
   ```bash
   npm run dev
   ```
